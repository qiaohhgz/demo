# 快速部署一个静态页面到GitHub


```git
> git init
> git add index.html
> git commit -m 'first commit'
> git remote add origin [URL]
> git push origin ph-pages
```

打开网页 http://[github账号名].github.io/[项目名字]/index.html

### 举例

假设github账号名为 qiaohhgz

假设项目名字为demo

打开网页 http://qiaohhgz.github.io/demo/index.html
